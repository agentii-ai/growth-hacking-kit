# Growth Hacking Kit Constitution
## Version 1.1.0

The Growth Hacking Kit provides templates and commands for growth campaign management.

### Working Directory
.growthkit/

### Core Commands
- /growthkit.plan - Create growth campaign plans
- /growthkit.tasks - Generate growth task breakdowns
- /growthkit.specify - Define growth campaign specifications
- /growthkit.implement - Implement growth initiatives
- /growthkit.constitution - View this constitution

### Principles
Growth Hacking Kit follows these core principles for managing growth campaigns and viral mechanics.
